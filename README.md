Gentle Dose – Smart Medication Reminder App

Gentle Dose is a healthcare mobile application developed using Flutter, Dart, and Firebase to help patients manage their medications while enabling caregivers to monitor treatment progress remotely.

The application provides medication reminders, medicine schedules, doctor appointment notifications, and caregiver monitoring. It supports two user roles: Patient and Caregiver. Firebase Authentication and Cloud Firestore were used for secure user management and real-time data synchronization.

Features
- Patient & Caregiver Authentication
- Medicine Schedule Management
- Medication Reminders
- Doctor Appointment Tracking
- Caregiver Dashboard
- Firebase Authentication
- Cloud Firestore Database
- English Language Support (Urdu planned)

Technologies
- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
